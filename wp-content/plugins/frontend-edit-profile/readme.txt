WordPress Plugin to show profile edit backend on posts or pages

== Description ==

Show profile edit backend on posts or pages through shortcode, without coding or hacks. Great for membership site.

<b>Demo:</b>This plugin already tested on <a href="http://localorganicfoodstore.com">organic food store</a> (register your self to try).
